Overview

The L2_LR_SSH product is SWOT’s KaRIn low-rate sea-surface-height product for oceanography. It provides sea surface height (SSH), sea surface height anomaly (SSHA), significant wave height, sigma0, wind speed, uncertainties, quality flags, and supporting geophysical and correction information. Each granule spans one spacecraft pass and is distributed as four NetCDF files.

This is a summary of the product documents Version C and Version D on PODAAC.

How to read the CRID

CRID pattern

P [I or G] [C or D] [minor]
  • P: fixed prefix
  • I: forward-processed
  • G: reprocessed
  • C or D: major version
  • Final character: minor release code

Examples

  • PIC0: Version C forward product
  • PGC0: Version C reprocessed product
  • PIC2: updated Version C forward product with resolved issues
  • PID0: Version D forward product
  • PGD0: Version D reprocessed product

L2_LR_SSH file set

Each pass-sized granule is distributed as four NetCDF files. The first three are on a geographically fixed 2 km swath-aligned grid; the fourth is on the native unsmoothed grid.

File Main purpose Sampling
Basic Core SSH / SSHA and quality information 2 km fixed swath-aligned grid
WindWave SWH, sigma0, wind speed, and model wave/wind context 2 km fixed swath-aligned grid
Expert Superset of Basic + WindWave with detailed corrections and ancillary fields 2 km fixed swath-aligned grid
Unsmoothed Near-native SSH / sigma0 fields on the center-beam native grid ~250 m posting, ~500 m resolution

The product latency stated in the product description is less than 3 days from data collection.

Practical use guidance

For a continuous LR SSH time series, use Version D whenever it exists. For dates before the Version D reprocessing/final coverage, fall back to the highest available Version C product. A practical sequence is: PGD0 → PID0 → PIC2 → PGC0/PIC0.

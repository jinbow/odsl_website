OCNG 489/689 — Study the Oceans from Space

An introduction to satellite oceanography spanning fundamentals (EM radiation, sensors, data products) through applications (ocean color, SST, winds, altimetry, SAR, gravity/laser, sea level/ice) and modern themes (AI, data fusion). This page provides a module-based entry point.

Tip: Each module card below links to a standalone HTML page. The resources page includes tools that help you to understand difficult concepts, data access workflows and other useful information.

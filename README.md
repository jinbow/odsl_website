# Ocean Dynamics from Space Website Template

This is a simple and elegant website template designed for the "Ocean Dynamics from Space" research group.

## Setup
1. **Download**: Copy all files into a local folder.
2. **Customize**:
   - Replace `assets/placeholder.jpg` and `assets/logo.png` with your own images.
   - Update text content in HTML files.
   - Adjust colors/fonts in `css/style.css` if desired.
3. **Deploy**:
   - Upload to a static hosting service like GitHub Pages or Netlify.
   - Optionally, integrate with a CMS (e.g., WordPress).

## Features
- Responsive design for all devices.
- High-definition image and animation support in the gallery.
- Lazy loading for performance.
